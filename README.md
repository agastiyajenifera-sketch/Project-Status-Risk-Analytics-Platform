# 🚀 ProActify AI – Project Status & Risk Analytics Platform

## 📌 Overview

**ProActify AI** is an AI-powered Project Status & Risk Analytics Platform designed to help project managers and teams monitor project progress, analyze performance, identify potential risks, and generate actionable insights through interactive dashboards and predictive analytics.

The platform transforms project data into meaningful visualizations, enabling organizations to make informed decisions and improve project success rates.

---

## ✨ Features

- 📊 Project Status Dashboard
- 📈 Real-time Progress Tracking
- 🤖 AI-Based Risk Analysis
- ⚠️ Risk Level Prediction
- 📉 Project Performance Analytics
- 📅 Timeline Monitoring
- 📋 Task Status Analysis
- 📌 KPI Visualization
- 📤 Report Generation
- 🎯 Interactive Charts and Graphs

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/yourusername/ProActify-AI.git
```

### Navigate to Project Folder

```bash
cd ProActify-AI
```

### Install Required Packages

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
streamlit run app.py
```
